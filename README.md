# LBIGD (Faction Chess IGD)

**체스-유사 격자 게임에서, 팩션별 말(행마/스탯/개수) + 맵(크기/장애물)을 설계변수로 두고 자동 밸런싱하는 IGD 프로젝트**

## 1. 무엇을 하는 프로젝트인가
이 레포의 목표는 아래를 자동으로 탐색하는 것입니다.

- 팩션별 말 설계: **행마 패턴 / 이동 / 사거리 / 공격력 / 체력 / 개수**
- 맵 설계: **맵 크기 / 장애물 밀도·패턴**

평가는 self-play 결과(승률/무승부/교전 거리 등)로 수행하고, Outer loop(ES)가 설계변수를 업데이트합니다.

## 2. 실행 (Quick Start)

### 1) 설치
Python(3.12 이상)이 설치되어 있어야 합니다. 패키지 관리자 `uv`를 사용합니다.

```bash
# uv 설치 (이미 있다면 패스)
pip install uv

# 필요한 라이브러리 한방에 설치
uv sync
```

### 2) 실행
설치가 끝났다면 바로 실행해 보세요.

```bash
# Windows
.venv/Scripts/python main.py

# Mac / Linux
.venv/bin/python main.py
```

### 3) 결과
`main.py`가 ES 최적화 루프를 돌리며, 후보 설계들의 통계와 최종 설계를 출력합니다.
